JLI LinkedIn Declutter

A CSS-only Chrome extension that hides and collapses distracting LinkedIn feed and notification clutter.

Single Purpose

JLI LinkedIn Declutter locally hides and collapses distracting LinkedIn feed and notification clutter with a packaged stylesheet so users can browse LinkedIn with less noise.

What It Does

  • Hides LinkedIn News and right-rail clutter.
  • Hides right-rail ad iframes.
  • Collapses promoted, sponsored, and suggested feed posts.
  • Hides job recommendation feed cards.
  • Collapses recommended follow modules.
  • Collapses reaction and analytics notifications.
  • Uses CSS-only hover-to-expand cards.

What It Does Not Do

  • Does not use LinkedIn APIs.
  • Does not run JavaScript on LinkedIn pages.
  • Does not observe, add, move, or rewrite LinkedIn DOM nodes.
  • Does not send messages or post content.
  • Does not click buttons, like posts, follow people, or connect with people.
  • Does not load remote code.
  • Does not transmit data to the developer or external servers.

Privacy Policy

Last updated: May 10, 2026

The packaged extension does not process LinkedIn page content with JavaScript. It applies local CSS rules in the browser to hide or collapse matching clutter, such as promoted posts, sponsored-looking cards, suggested posts, recommendation modules, reaction notifications, and analytics notifications.

The extension does not store configuration settings and does not request Chrome storage permission.

No analytics, advertising, tracking pixels, remote logging, or remote code are included. All CSS, HTML, and assets are packaged with the extension.

Permissions

Storage: Not requested.

LinkedIn host access: Required for https://www.linkedin.com/* so Chrome can apply the packaged stylesheet to LinkedIn pages. Access is limited to LinkedIn.

Deleting Data

The packaged extension stores no local settings. Remove the extension from Chrome to remove the stylesheet.
