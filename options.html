JLI Declutter

CSS-only LinkedIn decluttering. No JavaScript runs on LinkedIn pages.

What this extension does

The packaged extension injects one static stylesheet on LinkedIn. It does not read page text, add nodes, move nodes, set attributes, observe DOM changes, click buttons, call APIs, or send data anywhere.

Included rules

  • Collapse promoted/sponsored-looking feed cards.
  • Collapse social/reaction feed cards.
  • Hide job recommendation carousels.
  • Hide right-rail ad iframes and broad right-rail clutter.
  • Collapse reaction and analytics notifications.

Collapsed cards expand on hover.

Want custom toggles?

Use the Stylus CSS generator for a configurable stylesheet. That path is more flexible and keeps the extension runtime even smaller.

Debug tip

If a card is missed, LinkedIn probably changed its markup. The static stylesheet intentionally favors less code and less page interference over perfect matching.
